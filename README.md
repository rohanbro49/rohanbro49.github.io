## Hosting on GitHub Pages

To host this website on GitHub Pages:

1. Create a GitHub repository named `username.github.io` (replace `username` with your GitHub username)
2. Push this entire folder structure to the repository
3. The website will be live at `https://username.github.io`

### Setup Steps:

```bash
# Initialize git repository (if not already done)
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: Mark Institute Australia website"

# Add remote (replace with your repository)
git remote add origin https://github.com/username/username.github.io.git

# Push to GitHub
git branch -M main
git push -u origin main
```
